# DigiTools

A modern digital tools storefront built in React and Tailwind CSS. Select premium products, add them to the cart, and checkout with smooth toast notifications.

## Technologies used
- React.js
- Tailwind CSS
- React-Toastify
- JavaScript (ES6+)

## Features
- Product grid with 8 digital tool cards
- Cart functionality with add, remove, and checkout
- Toast alerts for add/remove/checkout actions
- Responsive design for desktop and mobile
- Product tags, pricing, and feature lists
